# 🌐 Portfolio Website

## 📌 Overview
This is a responsive portfolio website built using HTML and CSS. It demonstrates styling, layout, and responsive design.

## ⚙️ Setup Instructions
1. Download or clone the repository
2. Open index.html in any browser

## 🧱 Project Structure
- index.html → Structure
- style.css → Styling
- images/ → Assets
- screenshots/ → Output images

## 🎨 CSS Concepts Used
- CSS Selectors (element, class, ID)
- Flexbox layout
- Box model
- Hover effects
- Media queries (responsive design)

## 📱 Responsive Design
The website adjusts layout for mobile screens using media queries (max-width: 768px).

## 🧪 Testing
Tested on:
- Chrome
- Edge
- Mobile view

## 🚀 Features
- Clean UI design
- Responsive layout
- Interactive buttons
- Flexbox-based structure
